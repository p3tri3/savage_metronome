# Project Vision
- Desktop metronome application written in Rust.
- Educational project focused on learning:
- Clean architecture
- Modular Rust structure
- Test-driven development
- AI-assisted development workflows

# Core Functional Goals
- Accurate BPM timing
- Configurable time signatures
- Custom metronome sounds (user-provided WAV)
- Preset save/load system
- Expandable UI

# Non-Goals
- Cloud sync
- Mobile version
- Complex DAW integration
- Plugin system (unless later explicitly added)

# Architectural Intent
- Clear separation of domain logic from UI and audio backends.
- High testability.
- Feature-based modular structure.
- Maintainable code suitable for iterative AI collaboration.
